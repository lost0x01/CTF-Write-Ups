# CTF Write-Ups

Collection of Capture The Flag (CTF) write-ups and solutions from various competitions and platforms.

## About

This repository contains detailed write-ups for CTF challenges I've solved, including:
- Challenge descriptions
- Step-by-step solutions
- Tools and techniques used
- Key learnings and insights

## Contents

- **Web Security** - Web application vulnerabilities and exploitation
- **Cryptography** - Encryption, hashing, and cryptanalysis challenges
- **Reverse Engineering** - Binary analysis and decompilation
- **Forensics** - Digital forensics and data analysis
- **Pwn** - Binary exploitation and memory corruption
- **OSINT** - Open source intelligence gathering

## Competitions

Solutions from various CTF platforms:
- Capture The Flag competitions
- HackTheBox
- TryHackMe
- PicoCTF
- And more...

## How to Use

Each challenge has its own directory with:
- `README.md` - Challenge description and walkthrough
- Solution files
- Scripts and tools used
